# 💰 Student Budget Tracker

A simple and beginner-friendly **Budget Tracker web app** built using **HTML, CSS, and JavaScript (Frontend Only)**.
It helps students manage their income and expenses easily.

---

## 🚀 Features

* ➕ Add income and expense transactions
* 📊 Automatically calculates balance
* 📈 Shows total income and expenses
* 🗑️ Delete transactions
* 🎨 Clean and simple UI

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript (Vanilla JS)

---

## 📂 Project Structure

```
budget-tracker/
 └── index.html
```

*(All code is written in a single file for simplicity)*

---

## ▶️ How to Run

1. Download or clone the repository
2. Open `index.html` in your browser

That’s it! 🎉

## 📸 Preview

## 📌 How It Works

* User enters a description and amount
* Positive amount → Income
* Negative amount → Expense
* JavaScript updates:

  * Balance
  * Income total
  * Expense total
* Transactions are displayed dynamically
